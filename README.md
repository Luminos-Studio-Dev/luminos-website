# 🌌 Liminos Studio Dev | 公式ホームページ

このリポジトリは、Liminos Studio Dev チームの活動、プロジェクト、そしてメンバーを紹介するための公式ホームページのソースコードです。

**Museboard** や **Aether Status Bot** などの主要プロジェクトの進捗と、私たちの技術的な専門分野を紹介しています。

---

## 🚀 概要 (Features)

* **静的サイト構成:** HTML / CSS / Vanilla JavaScript を使用したシンプルな静的ページとして構築されています。
* **ダークモード/ライトモード:** ユーザーの好みに合わせてテーマを切り替える機能。
* **多言語対応 (i18n):** JavaScript を使用した日本語/英語のコンテンツ切り替え機能。
* **動的なプロジェクト一覧:** `assets/projects.json` ファイルからデータを取得し、プロジェクト一覧を動的に表示します。
* **コンタクト & メンバー紹介:** 独立したページ（`contact.html`, `members.html`）による詳細な情報提供。

---

## 🛠️ 技術スタック

| 分野 | 技術 | 備考 |
| :--- | :--- | :--- |
| **フロントエンド** | HTML5, CSS3 | クリーンでセマンティックなマークアップ |
| **インタラクティブ** | Vanilla JavaScript | フレームワーク不使用。ダークモード、i18n、データロードを担当 |
| **データ管理** | JSON | プロジェクト一覧 (`projects.json`) および翻訳データ (`i18n/*.json`) の管理 |
| **スタイリング** | CSS Variables | ダークモード/ライトモード切り替えを容易にするカスタムプロパティを使用 |
